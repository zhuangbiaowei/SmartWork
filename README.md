# SmartWork

AI 智能体协作平台

## 项目结构

```
SmartWork/
├── src/
│   ├── frontend/          # Electron + React 前端
│   └── backend/           # FastAPI + Python 后端
├── docs/                  # 项目文档
├── tests/                 # 测试文件
└── package.json          # 根项目配置
```

## 快速开始

### 安装依赖

```bash
# 安装根依赖
npm install

# 安装前端依赖
cd src/frontend && npm install

# 安装后端依赖
cd src/backend && pip install -r requirements.txt
```

### 开发模式

```bash
npm run dev
```
